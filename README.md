## RU Описание

ЧёЕдят - сайт, который позволяет делиться с миром и близкими своими кулинарными рецептами.
Он включает в себя:
- Простой интерфейс для работы с рецептами
- Полный функционал для публикации и поиска рецептов
- Удобная функция импорта рецептов из приложения ЧёПоесть для быстрой публикации
- Сохранение рецептов в отдельном файле, который можно легко импортировать в ЧёПоесть
- Возможность создавать приватные рецепты, доступные только по ссылке

Использованные технологии:
- Python 3.x
- Библиотеки:
  - Flask
  - SQLite3
  - HashLib
  - Sys и Os
  - re
  - Random

## RU Ожидания от проекта
- Создания удобного средства обмена рецептами
- Расширение функционала: комментирование рецептов, возможность добавлять в избранное рецепты и подписываться на авторов
- Возможное добавление поддержки других языков

## EN Description

WhatPplEat - site, which allows people to share their recipes with others.
It includes:
- Simple interface for working with recipes
- Full functionality for publishing and searching recipes
- Convenient function of importing recipes from WhatTheEat app for faster publication
- Saving recipes in a file, that could be easily imported into WhatTheEat
- Ability to create private recipes, which are available only trough link

Used tools:
- Python 3.x
- Libraries:
  - Flask
  - SQLite3
  - HashLib
  - Sys и Os
  - re
  - Random

## EN Expectations from the project
- Creating a comfortable toll for sharing recipes
- Implementation of functions: commenting recipes, ability to put recipes into faviurite category and subscribing to authors
- Implementation of other languages support
